# VEGETABLE CROPS Image Classification

## Project Overview
This project focuses on the cultivation of various vegetable crops including cabbage, cucumber, eggplant, sweet potato, spinach, chili, string beans, tomato, okra, kangkong, pechay, saluyot, alugbati, patola, upo, kalabasa, gabi, singkamas, munggo, and patani.

The goal is to produce fresh, nutritious vegetables for household consumption and local markets while promoting sustainable farming practices. The project includes land preparation, planting, crop management, pest control, harvesting, and marketing.

By growing a diverse range of leafy vegetables, fruit vegetables, legumes, and root crops, the project supports food security, improves income opportunities, and encourages year-round production. 🌿🌾

## Plant Species Section
1. Cabbage
Brassica oleracea
Common Name: Cabbage
Scientific Name: Brassica oleracea
Description: A leafy vegetable forming a dense, round head of tightly packed leaves. It thrives in cool climates and is commonly used in salads, soups, and stir-fried dishes.

2. Cucumber
Cucumis sativus
Common Name: Cucumber
Scientific Name: Cucumis sativus
Description: A creeping vine plant producing long, green fruits with high water content. It is widely consumed fresh, pickled, or in salads.

3. Eggplant
Solanum melongena
Common Name: Eggplant
Scientific Name: Solanum melongena
Description: A warm-season vegetable producing glossy purple (sometimes white or green) fruits. It is commonly used in stews, grilling, and sautéed dishes.

4. Sweet Potato
Ipomoea batatas
Common Name: Sweet Potato
Scientific Name: Ipomoea batatas
Description: A root crop with sweet-tasting tubers, rich in carbohydrates and vitamins. It grows well in warm climates.

5. Spinach
Spinacia oleracea
Common Name: Spinach
Scientific Name: Spinacia oleracea
Description: A leafy green vegetable rich in iron and vitamins. It grows best in cool weather and is used in salads and cooked dishes.

6. Chili
Capsicum annuum
Common Name: Chili
Scientific Name: Capsicum annuum
Description: A fruit-bearing plant producing spicy peppers. It is widely used as a seasoning and spice.

7. String Beans
Vigna unguiculata
Common Name: String Beans (Yardlong Bean)
Scientific Name: Vigna unguiculata subsp. sesquipedalis
Description: A climbing plant producing long, slender green pods. It thrives in warm climates.

8. Tomato
Solanum lycopersicum
Common Name: Tomato
Scientific Name: Solanum lycopersicum
Description: A widely cultivated fruit vegetable used in sauces, salads, and cooking.

9. Okra
Abelmoschus esculentus
Common Name: Okra
Scientific Name: Abelmoschus esculentus
Description: A warm-season vegetable known for its edible green pods, commonly used in soups and stews.

10. Kangkong
Ipomoea aquatica
Common Name: Kangkong (Water Spinach)
Scientific Name: Ipomoea aquatica
Description: A semi-aquatic leafy vegetable commonly grown in tropical regions.

11. Pechay
Brassica rapa
Common Name: Pechay (Bok Choy)
Scientific Name: Brassica rapa subsp. chinensis
Description: A leafy vegetable with white stalks and green leaves, commonly used in Asian cuisine.

12. Saluyot
Corchorus olitorius
Common Name: Saluyot (Jute Mallow)
Scientific Name: Corchorus olitorius
Description: A leafy vegetable known for its slightly slimy texture when cooked.

13. Alugbati
Basella alba
Common Name: Alugbati (Malabar Spinach)
Scientific Name: Basella alba
Description: A fast-growing vine with thick, glossy leaves used in soups and stir-fries.

14. Patola
Luffa acutangula
Common Name: Patola (Ridge Gourd)
Scientific Name: Luffa acutangula
Description: A climbing vine producing ridged green fruits used in cooking.

15. Upo
Lagenaria siceraria
Common Name: Upo (Bottle Gourd)
Scientific Name: Lagenaria siceraria
Description: A vine plant producing light green elongated fruits used in soups.

16. Kalabasa
Cucurbita moschata
Common Name: Kalabasa (Squash)
Scientific Name: Cucurbita moschata
Description: A creeping plant producing large orange fruits rich in vitamins.

17. Gabi
Colocasia esculenta
Common Name: Gabi (Taro)
Scientific Name: Colocasia esculenta
Description: A root crop grown for its starchy corms and large heart-shaped leaves.

18. Singkamas
Pachyrhizus erosus
Common Name: Singkamas (Jicama)
Scientific Name: Pachyrhizus erosus
Description: A root vegetable with crisp, sweet white flesh often eaten raw.

19. Munggo
Vigna radiata
Common Name: Munggo (Mung Bean)
Scientific Name: Vigna radiata
Description: A legume producing small green beans used in soups and desserts.

20. Patani
Phaseolus lunatus
Common Name: Patani (Lima Bean)
Scientific Name: Phaseolus lunatus
Description: A climbing plant producing flat edible beans used in various dishes.
